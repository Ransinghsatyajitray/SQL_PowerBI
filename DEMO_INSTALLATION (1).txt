SQL SERVER INSTALLATION:
	INSTANCE:	A SUCESSFUL INSTALLATION OF SQL SERVER. 
			TYPES OF INSTANCES:
			DEFAULT INSTANCE	:	INSTANCE NAME = MSSQLSERVER. USED IN PRODUCTION.	
							SERVERNAME = COMPUTERNAME
				
			NAMED INSTANCE		:	INSTANCE NAME = defined @ installation. USED IN DEV, TEST
							SERVERNAME = COMPUTERNAME\INSTANCENAME
	DOWNLOAD SQL SERVER FROM WWW.SQLSCHOOL.COM/DOWNLOADS
	DOUBLE CLICK SETUP FILE > SELECT DATABASE ENGINE > SPECIFY INSTANCE NAME > ADD CURRENT USER > INSTALL
